# Skills.sh 一键安装

通过 [skills.sh](https://skills.sh/zeij-drive/deobf) 安装 `deobf-all` 调度器，自动识别所有支持的 agent 平台。

如果你需要同时预装全部 26 个子 skill，建议使用 [curl|bash 一键全量安装](/quickstart#方式%E2%80%890curl--bash-%E4%B8%80%E9%94%AE%E5%85%A8%E9%87%8F%E5%AE%89%E8%A3%85%E6%9C%80%E6%8E%A8%E8%8D%90)。

## 安装

```bash
npx skills add zeij-drive/deobf -g -y
```

安装完成后，调度 skill 将被注册到以下位置：
- 全局 skill 目录：`~/.agents/skills/deobf-all/`
- 所有已安装的 agent 平台（Claude Code、Cursor、Codex、Gemini CLI 等）

## 工作原理

`npx skills add` 执行时：

1. 从 GitHub 仓库 `zeij-drive/deobf` 拉取 `deobf-all/SKILL.md`
2. 安装到本地 `~/.agents/skills/deobf-all/` 目录
3. 自动扫描并注册到已安装的 agent 平台

首次调用 `/deobf-all` 时，调度器会先对目标分类，再按需加载相关子 skill。

## 与 curl|bash 一键安装的区别

| 特性 | Skills.sh（调度器） | curl|bash（全量） |
|------|-------------------|--------------------|
| 安装内容 | 仅调度器 | 调度器 + 全部 26 子 skill |
| 首次调用 | 自动 read_skill 拉取 | 已预装，无需等待 |
| 命令数 | 1 条 | 1 条（curl pipe） |
| 离线可用 | 否 | 否（首次需网络） |
