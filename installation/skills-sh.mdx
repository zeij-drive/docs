# Skills.sh 一键安装

通过 [skills.sh](https://skills.sh/zeij-drive/deobf) 一条命令安装，自动识别所有支持的 agent 平台。

## 安装

```bash
npx skills add zeij-drive/deobf -g -y
```

安装完成后，调度 skill 将被注册到以下位置：
- 全局 skill 目录：`~/.agents/skills/deobf-all/`
- 所有已安装的 agent 平台（Claude Code、Cursor、Codex、Gemini CLI 等）

## 工作原理

`npx skills add` 执行时：

1. 从 GitHub 仓库 `zeij-drive/deobf` 拉取 `deobf-all/SKILL.md`
2. 安装到本地 `~/.agents/skills/deobf-all/` 目录
3. 自动扫描并注册到已安装的 agent 平台

首次调用 `/deobf-all` 时，调度器会自动通过 `read_skill` 拉起全部 9 个子 skill。

## 与安装脚本的区别

| 特性 | Skills.sh 一键安装 | 克隆仓库 + 安装脚本 |
|------|-------------------|---------------------|
| 命令数 | 1 条 | 2-3 条 |
| 离线可用 | 否 | 是 |
| 预装子 skill | 否（首次调用时自动拉起） | 是（同时安装全部 9 个） |
| 自动注册 agent | 是 | 部分（取决于脚本实现） |
| 网络要求 | 需要访问 GitHub | 需要访问 GitHub + npm |

<Info>
如果你希望离线可用，或需要同时预装全部 9 个子 skill，请使用[克隆仓库安装](/installation/auto-install)方式。
</Info>
