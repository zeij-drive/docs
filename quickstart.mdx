# 快速开始

## 方式 0：curl | bash / Windows 一键全量安装（最推荐 🌟）

一条命令安装**全部 26 个子 skill + deobf-all 调度器**，无需克隆仓库：

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/zeij-drive/deobf/master/install.sh | bash
    ```
  </Tab>
  <Tab title="Windows (PowerShell)">
    ```powershell
    $ErrorActionPreference = "Stop"; $url = "https://raw.githubusercontent.com/zeij-drive/deobf/master/install.bat"; $dst = Join-Path $env:TEMP "deobf.bat"; try { [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; Invoke-WebRequest -UseBasicParsing $url -OutFile $dst } catch { curl.exe --ssl-no-revoke -fL --retry 3 -o $dst $url; if ($LASTEXITCODE -ne 0) { throw "Download failed (curl exit code $LASTEXITCODE)" } }; Unblock-File -LiteralPath $dst; cmd /c $dst
    ```
  </Tab>
</Tabs>

安装后首次调用 `/deobf-all` 时会自动按需加载子 skill。如需**完整预装全部 26 个子 skill**，上面的 PowerShell 命令即可全量安装，无需克隆仓库。
- 自动注册到所有已安装的 agent 平台

## 方式 1：Skills.sh 一键安装（安装调度器）

```bash
npx skills add zeij-drive/deobf --skill deobf-all -g -y
```

> 通过 [skills.sh](https://skills.sh/zeij-drive/deobf) 安装 `deobf-all` 调度 skill。首次调用时会自动拉起子 skill。

## 方式 2：克隆仓库 + 安装脚本

<Steps>

**克隆仓库**

```bash
git clone https://github.com/zeij-drive/deobf.git
cd deobf
```

**运行安装脚本**

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
  </Tab>
  <Tab title="Windows">
    ```bat
    install.bat
    ```
  </Tab>
</Tabs>

安装脚本将自动完成：
- 通过 `npx skills add` 全局安装全部 **26 个子 skill**
- 通过 `npx skills add zeij-drive/deobf --skill deobf-all -g -y` 全局安装 `deobf-all` 调度 skill
- 使用 `--local` 时安装到当前 workspace 的 `.agents/skills/` 路径

**开始使用**

在支持的 Agent 平台中直接调用：

<CodeGroup>
  ```text Reasonix / Claude Code
  /deobf-all
  ```
  ```python 编程方式
  run_skill(name="deobf-all")
  ```
</CodeGroup>

</Steps>

## 安装选项

安装脚本支持以下参数：

| 参数 | 说明 |
|------|------|
| `--global` | 全局安装技能（默认） |
| `--local` | 将技能安装到当前 workspace，不影响全局目录 |
| `--dry-run` | 只预览将要安装的技能，不真正执行安装 |
| `--force` | 强制重新安装已存在的 skill |
| `--no-deps` | 仅安装调度器，跳过子 skill |
| `--help` | 显示帮助信息 |

```bash
# 安装到当前项目（不影响全局）
./install.sh --local

# 预览安装内容
./install.sh --dry-run

# 仅安装调度器
./install.sh --no-deps
```

<Tip>
如果是首次使用，建议先运行 `--dry-run` 预览，确认无误后再执行安装。
</Tip>
