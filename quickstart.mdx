# 快速开始

## 方式 0：Skills.sh 一键安装（最推荐）

```bash
npx skills add zeij-drive/deobf -g -y
```

这一条命令即可：
- 自动发现仓库中的 `deobf-all` 调度 skill
- 安装到全局 skill 目录（`~/.agents/skills/deobf-all/`）
- 自动注册到所有已安装的 agent 平台（Claude Code、Cursor、Codex、Gemini CLI 等）

<Info>
在 [skills.sh/zeij-drive/deobf](https://skills.sh/zeij-drive/deobf) 查看详情和安装量。
</Info>

安装调度 skill 后，首次调用 `/deobf-all` 时它会自动 `read_skill` 拉起全部 9 个子 skill。如果你希望离线安装或同时预装所有子 skill，请用下方的方式 1。

## 方式 1：克隆仓库 + 安装脚本

<Steps>

**克隆仓库**

```bash
git clone https://github.com/zeij-drive/deobf.git
cd deobf
```

**运行安装脚本**

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
  </Tab>
  <Tab title="Windows">
    ```bat
    install.bat
    ```
  </Tab>
</Tabs>

安装脚本将自动完成：
- 通过 `npx skills add` 全局安装全部 **9 个子 skill**
- 将 `deobf-all` 调度 skill 复制到 `~/.agents/skills/`
- 如果在当前 workspace 中运行，也会安装到本地 skill 路径

**开始使用**

在支持的 Agent 平台中直接调用：

<CodeGroup>
  ```text Reasonix / Claude Code
  /deobf-all
  ```
  ```python 编程方式
  run_skill(name="deobf-all")
  ```
</CodeGroup>

</Steps>

## 安装选项

安装脚本支持以下参数：

| 参数 | 说明 |
|------|------|
| `--local` | 将技能安装到当前 workspace，不影响全局目录 |
| `--dry-run` | 只预览将要安装的技能，不真正执行安装 |
| `--help` | 显示帮助信息 |

```bash
# 安装到当前项目（不影响全局）
./install.sh --local

# 预览安装内容
./install.sh --dry-run
```

<Tip>
如果是首次使用，建议先运行 `--dry-run` 预览，确认无误后再执行安装。
</Tip>
