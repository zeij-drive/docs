# Documentation agent instructions

## 项目上下文
- deobf-all 是 AI Agent 统一反混淆技能套件，一条命令加载 9 个专业反混淆/逆向子 skill
- 目标用户：安全研究人员、逆向工程师、CTF 选手
- 文档语言：中文（代码/技术术语保留英文原文）

## 代码示例
- Shell 命令统一使用 bash 语法
- 安装命令区分 macOS/Linux 和 Windows 平台
- 在代码块中标明语言类型

## API 文档标准
- 每个 Skill 参考页必须包含：来源仓库、安装量、优先级、支持的混淆类型
- 使用表格列出技术细节
- 提供路由触发条件说明

## 风格与格式
- 面向有基础逆向工程经验的开发者，不做过度简化
- 使用主动语态和第二人称（"你"）
- 引用 UI 元素时使用加粗：**Settings** 按钮
- 日期格式：ISO 8601 (YYYY-MM-DD)

## 内容要求
- 指南页面在需要前置条件时添加前置条件章节
- 各页面末尾添加相关文档的交叉链接
- 为常见故障场景提供故障排除章节
