# deobf-all 简介

**deobf-all** 是一个 Agent Skill，充当反混淆工作流的统一入口。一条命令 `npx skills add zeij-drive/deobf -g -y` 即可安装，调用一次 `/deobf-all`，它就会自动完成全部调度。

<Steps>
  **加载**
  
  同时加载全部 9 个子 skill 到 agent 上下文中

  **分类**
  
  对你的目标进行分类（原生二进制、JavaScript、字节码、CTF、未知类型）

  **路由**
  
  路由到最优 skill 组合，针对具体的混淆类型

  **验证**
  
  引导完整的反混淆工作流，确认输出正确性
</Steps>

## 为什么选择 deobf-all？
