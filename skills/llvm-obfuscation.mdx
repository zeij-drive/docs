# llvm-obfuscation

<Info>
**来源**: [gmh5225/awesome-llvm-security](https://github.com/gmh5225/awesome-llvm-security) &nbsp;·&nbsp; **安装量**: 65 &nbsp;·&nbsp; **优先级**: P3
</Info>

LLVM 混淆分析 Skill，专注于 OLLVM 等基于 LLVM 的混淆技术。

## 核心能力

- OLLVM 控制流扁平化分析
- 虚假分支（Bogus CF）识别
- 指令替换（Instructions Substitution）还原
- 基于 LLVM Pass 的反混淆

## 适用场景

- OLLVM / Armarines 混淆的二进制
- 基于 LLVM 的自定义混淆器
- 需要从 IR 级别还原控制流
