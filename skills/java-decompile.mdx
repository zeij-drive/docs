# java-decompile

<Info>
**来源**: [quarkusio/quarkusdev-skills](https://github.com/quarkusio/quarkusdev-skills) &nbsp;·&nbsp; **安装量**: 37 &nbsp;·&nbsp; **优先级**: P2
</Info>

Java 字节码反编译 Skill，支持 CFR、Procyon、Fernflower 后端，将 JAR/class 文件还原为可读 Java 源码。

## 核心能力

- JAR/class 字节码反编译
- 多后端支持（CFR / Procyon / Fernflower）
- 方法签名与泛型还原
- 依赖库源码查看

## 适用场景

- 分析第三方依赖库的实现细节
- 追溯方法调用链
- 查找硬编码凭据或配置
- 理解无源码的 Java 类行为
