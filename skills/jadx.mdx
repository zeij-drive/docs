# jadx

<Info>
**来源**: [brownfinesecurity/iothackbot](https://github.com/brownfinesecurity/iothackbot) &nbsp;·&nbsp; **安装量**: 316 &nbsp;·&nbsp; **优先级**: P2
</Info>

Android APK 反编译 Skill，将 DEX 字节码转换为可读的 Java 源码，支持资源解析和混淆还原。

## 核心能力

- DEX → Java 源码反编译
- APK 资源文件解析
- AndroidManifest.xml 分析
- 混淆代码还原与重命名建议

## 适用场景

- APK 逆向分析与安全审计
- 查找硬编码 API Key / 凭据
- 分析第三方 SDK 行为
- 恶意代码检测
