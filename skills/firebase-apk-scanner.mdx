# firebase-apk-scanner

<Info>
**来源**: [trailofbits/skills](https://github.com/trailofbits/skills) &nbsp;·&nbsp; **安装量**: 3.5K &nbsp;·&nbsp; **优先级**: P3
</Info>

APK 的 Firebase 配置安全扫描 Skill，自动检测 Firebase 数据库、存储桶的开放权限。

## 核心能力

- Firebase URL 自动提取
- 实时数据库开放权限检测
- Firebase Storage 公开访问检测
- 安全隐患报告生成

## 适用场景

- APK 安全审计中的 Firebase 检测
- 检查 Firebase 配置是否暴露敏感数据
- 移动应用后端安全评估
