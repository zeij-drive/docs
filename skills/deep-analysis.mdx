# deep-analysis

<Info>
**来源**: [cyberkaida/reverse-engineering-assistant](https://github.com/cyberkaida/reverse-engineering-assistant) &nbsp;·&nbsp; **安装量**: 345 &nbsp;·&nbsp; **优先级**: P2
</Info>

深度逆向工程分析 Skill，专注于对未知目标进行全面分类和定向分析。

## 核心定位

当目标类型不明确、需要先进行分类再决定分析方向时，此 Skill 是优先选择。

## 分析能力

### 目标分类
- 原生二进制 vs 托管代码 vs 脚本
- 保护方案识别
- 混淆家族判断

### 全面侦察
- 文件格式与结构分析
- 导入/导出表审查
- 字符串分析
- 熵值检测（判断是否加壳/加密）

### 分析路径推荐
根据分类结果，推荐进一步的分析方向和对应的子 Skill 组合。

## 使用模式

| 模式 | 说明 |
|------|------|
| 分类先行 | 对未知目标先执行分类，再路由到专业 Skill |
| 辅助增强 | 为其他 Skill 提供更深入的上下文分析 |

## 路由触发条件

- 目标类型完全未知
- 之前的分析方向受阻，需要重新分类
- 需要更深入地理解目标结构后再继续
